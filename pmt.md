## Overview
I need the app to be working in the following way.

- There is a credit system in the application. each user will have a credit wallet.
- When the website is generated, it can be viewed only in the editor. Users should be blocked from viewing the source code of the generated site.
- Only when the user pays for any of the publishing plans, it can be opened as a link
    - Basic plan 500 Credits (Can publish only with our domain name -> like <user-provided-name>.genweb.in)
    - Single Page plan 2000 Credits (Can publish with user provided domain - buy from us for free or can connect with the domain they already have)
    - Multi Page plan 3000 Credits (Can publish with user provided domain - buy from us for free or can connect with the domain they already have)
- Several activities will consume the credits, generating a single page website will cost 200 credits, multipage will cost 400 credits, redesigning a section with prompt in the editor will cost 50 credits.
- Users can buy credits through our payment gateway (Razorpay)
